# Cricket Match Management System

A web-based cricket match management and live scoring application built using HTML, CSS, and JavaScript.

## Features

- Schedule cricket matches
- Live score updates
- Automatic target calculation
- Winner declaration
- Match history tracking
- Over and wicket management

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
index.html
schedule.html
scoreboard.html
history.html
styles.css
script.js
```

## Future Enhancements

- Toss functionality
- Player-wise scorecard
- Run rate calculation
- Team logos
- Dark mode
- Local storage support
